## 📌 Course Overview

This repository contains all lab submissions for **Database Systems-I (CSC371)**. The labs cover:

- Microsoft Access database design and implementation.
- Database keys, input masks, and validation rules.
- Forms, Reports, and Relationships.
- SQL (DDL and DML commands).
- Advanced queries and operators.

---

## 🛠️ Topics Covered

| Lab # | Title |
|-------|-------|
| Lab 01 | Introduction to Database and Tool Usage |
| Lab 02 | Database Keys, Format, Caption, Input Masks and Validation Rules |
| Lab 03 | Designing of Forms, Reports and Creation of Relationships |
| Lab 04 | Relationships, Simple & Advanced Queries |
| Lab 05 | Custom Calculations in Queries and Aggregate Functions |
| Lab 06 | Importing/Exporting Data, Make-Table & Union Queries |
| Lab 07 | Introduction to SQL - Database, Table Design & Data Insertion |
| Lab 08 | Implementation of DDL Commands in SQL |
| Lab 09 | Implementation of DML Commands in SQL |
| Lab 10 | Implementation of Different Types of Operators in SQL |
